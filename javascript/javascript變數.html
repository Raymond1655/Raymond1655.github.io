變數

   

㊙

JavaScript的變數型態包含Number,Boolean,String，可以使用typeof()函數來得到變數型態，Number可分為int與float。 說明如下:


           Number: 包含整數實數 >>    typeof(1)    typeof(3.14)



           Boolean: 包含true跟false >>    typeof(true)    typeof(false)



           String: 使用單引號或雙引號 >>    typeof('1')    typeof("1")



   

㊟

定義變數時，儘可能給予初始值，如此可以在之後再看程式碼時了解定義的變數型態。例如:


           let s = " " ;



           let a = [ ] ;



           let b = true ;





宣告

   

㊙

JavaScript不需要宣告變數型態。


           使用const關鍵字，定義常數，宣告的變數無法被改變 >>    const pi = 3.14159;



           使用var關鍵字 >>    var age = 18;



           使用let關鍵字 >>    let school = "NKUST";



           兩者的差異是let只作用於區塊 { } 內，區塊外無法使用let宣告的變數。而var在整個函數內使用。 


var age = 18;
if (true){
    let school = "NKUST";
}
   

㊟

我們應多使用let與const，避免使用var。因為let為區域變數，僅可用於被定義的區塊，而var則否，若名稱相同則會混淆。





Number

   

㊙

在宣告時會自動判斷是整數(int)或實數(float)，此外尚可將 16進位(0x開頭)或8進位(0開頭)或指數(E)轉成十進位數字。>>


           console.log(.12); // 0.12


           console.log(0b111); // 二進位:7


           console.log(0o12); // 八進位:10


           console.log(0x12); // 十六進位:18


           console.log(2E2); // 科學記號:100





   

㊮

若想將十進位數字傳換成其他進位，使用 toString ( n ) 。

let n = (100).toString(2);
console.log("100的二進位 =", n); // 1100100
let n = (100).toString(4);
console.log("100的四進位 =", n); // 1210
let n = (100).toString(8);
console.log("100的八進位 =", n); // 144
let n = (100).toString(16);
console.log("100的十六進位 =", n); // 64
   

㊮

若想將其他進位傳換成十進位，使用parseInt(num, n)。

let n = parseInt("1100100", 2);
console.log("二進位1100100 =", n); // 100
let n = parseInt("1210", 4);
console.log("四進位1210 =", n); // 100
let n = parseInt("144", 8);
console.log("八進位144 =", n); // 100
let n = parseInt("64", 16);
console.log("十六進位64 =", n); // 100
   

㊮

NaN表示Not a Number，在計算錯誤時出現。

console.log("a"-1); // NaN
   

㊮

整數的範圍介於-253 ~ 253之間(Number.MAX_SAFE_INTEGER)，而實數數字太大，懶得打， 可用MAX_VALUE與MIN_VALUE得到。

Number.MAX_VALUE // 1.7976931348623157e+308
Number.MIN_VALUE // 5e-324
   

㊮

如果數字超過上述之範圍，則用無限大(Infinity與-Infinity)來表示。

Number.MAX_VALUE + 1.0E308 // Infinity
-Number.MAX_VALUE - 1.0E308 // -Infinity
   

㊮

可以使用Number ( ) 函數將其他型態轉為數值型態。

var a = "12";
console.log(typeof(a)); // string
console.log(typeof(Number(a))); // number




Boolean

   

㊙

只有true跟false兩個值。


   

㊮

使用Boolean ( ) 函數來轉型。

var isYes = 1;
console.log(typeof(isYes)); // number
console.log(typeof(Boolean(isYes))); // boolean
console.log(Boolean(isYes)); // true
isYes = 0; // 0
console.log(Boolean(isYes)); // false
   

㊮

字串"false"，轉型後還是true。

var c = "false";
console.log(Boolean(c)); // true
   

㊮

使用Boolean ( ) 函數來轉型。

var d = undefined;
console.log(typeof(d)); // undefined
console.log(Boolean(d)); // false
   

㊟

使用Boolean ( ) 函數來轉型態，0跟1代表不同的值，會得到false有 : 數值為0或是NaN 、 空字串 、 null或undefined





String

   

㊮

使用單引號或雙引號表示，可以一起使用。

console.log("The car is 'so' big."); // The car is 'so' big.
   

㊮

使用escape character，例如:\b, \t, \n, \", \', \\等。

console.log('The car is \t\'so\'\t big.'); // The car is 	'so'	 big.
   

㊮

使用String ( ) 或toString ( ) 函數轉換型態，或是使用 + 轉換。

var s1 = String(true);
console.log(s1); // true
var s2 = toString(false);
console.log(typeof(s2)); // string
typeof("She weights" +45 + "kg."); // "string"
